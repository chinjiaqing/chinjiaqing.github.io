## 依赖 JRE8 安装

[https://www.oracle.com/java/technologies/downloads/#java8](https://www.oracle.com/java/technologies/downloads/#java8)

安装完毕后，将 `jrexxx_xxx\bin` 添加到`PATH`环境变量。

## 生成证书签名

```bash
keytool -genkey -alias testalias -keyalg RSA -keysize 2048 -validity 36500 -keystore test.keystore
```

- testalias 证书别名
- test.keystore 证书文件名
- 36500 证书有效期，表示100年

## 查看证书信息

```bash
keytool -list -v -keystore test.keystore
```

## 新版本不支持MD5查看的解决办法

### 1. 导出证书临时文件

```bash
keytool -exportcert -alias testalias -keystore test.keystore -file temp.cer
```

### 2. 计算MD5

```bash
certutil -hashfile temp.cer MD5
```

## 查看公钥

```bash
keytool -list -rfc -keystore test.keystore -alias testalias
```
